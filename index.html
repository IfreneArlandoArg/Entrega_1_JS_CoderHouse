Entrega 1 - CoderHouse

IFRENE, Arlando A

Simulador compra de indumentaria (ropas para hombre).

Funcionalidades

  • Elegir producto
    1. El sistema muestra un listado de productos con precios.
    2. El usuario selecciona un producto y confirma (Ok).
    3. El sistema pregunta talla del producto seleccionado.
    4. El usuario elige talla y confirma (Ok).
    5. El sistema pregunta por color del producto seleccionado.
    6. El usuario elige color y confirma (Ok).
    7. El sistema pregunta si quiere seguir comprando.
    8. El usuario elige si quiere seguir comprando o no.
    9. Si el usuario elige seguir comprando, el sistema redirige a la interfaz inicial de Elegir Producto.
    10. Si el usuario elige no seguir comprando, el sistema redirige a la interfaz principal (menu).

  • Hacer check-out
    1. Si el carrito está vacío, el sistema muestra un mensaje de error y agradece.
    2. Si hay productos en el carrito:
      • El sistema pide al usuario ingresar su nombre completo para la facturación.
      • El usuario ingresa su nombre completo y confirma (Ok).
      • El sistema pide al usuario ingresar su DNI.
      • El usuario ingresa su DNI y confirma (Ok).
      • El sistema pide dirección de entrega.
      • El usuario ingresa dirección de entrega y confirma (Ok).
      • El sistema pregunta teléfono de contacto.
      • El usuario ingresa teléfono de contacto y confirma (Ok).
      • El sistema informa de la cantidad que está a punto de pagar.
      • El usuario confirma (Ok).
      • El sistema confirma el pago.
      • El sistema muestra la factura con nombre, DNI del cliente, listado de productos comprados y total de la venta.
      • El sistema muestra un mensaje de agradecimiento.
      • El sistema guarda la venta (cliente + carrito), resetea el carrito (para la próxima venta) y redirige a la interfaz principal (menu).
